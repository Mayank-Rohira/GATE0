# GATE0: Features List

### 🎨 Design & Interaction
- **Deep Space Terminal UI**: A high-fidelity, futuristic dark theme.
- **Tactile Feedback**: Standardized components (TouchableOpacity) for reliable mobile interaction.
- **Animations**: Fluid spring animations and shimmer effects for loading states.
- **Zero Emojis**: Strictly professional, minimalist aesthetic.

### 🔑 Authorization Systems
- **Rich QR Schema**: Tokens embed full metadata (Visitor Name, Service, Resident Name, Unit Number).
- **One-Time Passcodes**: Securely generated strings for every entry request.
- **Role-Based Access**: Specialized dashboards for Residents, Guards, and Visitors.

### 🛡 Security & Audit
- **Real-time Logging**: Every entry attempt (Approved or Denied) is recorded in the system.
- **CSV Export**: Advanced UTF-8 compatible export with BOM for Excel/Sheets.
- **Deny & Flag**: Guards can reject suspicious entries, removing them from active passes.

### ⚙️ Developer Experience
- **Mobile-POV Optimized**: Easy IP configuration for testing on physical devices.
- **Modular Project Structure**: Clean separation of `backend/`, `frontend/`, and `docs/`.
